任何句子都是由主(Subject)和动词(Verb)组成，就是说**什么**在**干什么**。注意这里并不是有**主谓宾**组成。

有时候省略了主语，就成了**祈使句**，其实时省略了 You should



**名词做主语**

就是世间的万物，都可以是描述的对象，什么是什么，什么干什么

>The children is apt to tell lies



五大句型，

1. 主语 + 完全不及物动词
2. 主语 + 不完全及物 + 表语
3. 主语 + 完全及物 + 宾语
4. 主语 + 不完全及物 + 宾语 + 宾补
5. 主语 + 授予动词 + 间接宾语（授予对象） + 直接宾语（被授予的对象）
